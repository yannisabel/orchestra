### Custom Library

<Canvas of={IconStories.CustomLibrary} />

#### With Hardcoded SVGs

To use a custom icon library with hardcoded SVGs, register it before rendering:

```typescript
import { registerIconLibrary } from '@orchestra-kit/core'

registerIconLibrary('myIcons', {
  resolver: (name) => {
    const icons: Record<string, string> = {
      'star': '<svg viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">...</svg>',
      'heart': '<svg viewBox="0 0 24 24" xmlns="http://www.w3.org/2000/svg">...</svg>',
    }
    return icons[name] ?? ''
  }
})
```

#### From an External Package

For production use, import icons from an external package:

```typescript
import { registerIconLibrary } from '@orchestra-kit/core'
import * as Icons from '@your-org/icons-library'

registerIconLibrary('externalIcons', {
  resolver: (name) => {
    const iconExport = (Icons as Record<string, any>)[name]
    return typeof iconExport === 'string' ? iconExport : ''
  }
})
```

Then use it:

```html
<orchestra-icon name="star" library="externalIcons" fill="red" size="24px"></orchestra-icon>
```
