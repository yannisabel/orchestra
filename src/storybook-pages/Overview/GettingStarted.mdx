# Getting Started

From here you can start using the UI kit that follows Orchestra Design System principles.

## Installation

Install it as a package

</Box>
</Unstyled>

```
npm install orchestra-kit
```

<Box className="dark">

## Usage

### Notations

Notations are tokens, helpers, class utilities and all things that can help build components.

Import them all:

</Box>
</Unstyled>

```css
// all Notations can be used
@import 'orchestra-kit/dist/Notations';
// all Notations from a specific part can be used
@import 'orchestra-kit/dist/Notations/spaces';
```

<Unstyled>
<Box className="dark">

Usage of Notation token:

</Box>
</Unstyled>

```css
@import 'orchestra-kit/dist/Notations/spaces';

.custom-class {
  // the intellisense of the text editor should suggest available tokens
  padding: $space-3;
}
```
<Unstyled>
<Box className="dark">

Usage of useful mixins:

</Box>
</Unstyled>

```css
@import 'orchestra-kit/dist/Notations/mixins/breakpoints';
@import 'orchestra-kit/dist/Notations/fonts';

.custom-class {
  font-size: $fs-3;

  @include breakpoint(small) {
    font-size: $fs-5;
  }
}
```
<Unstyled>
<Box className="dark">

Importing utility classes.

</Box>
</Unstyled>

```css
// import them all 
@import 'orchestra-kit/dist/Notations/utility-classes';
// import only dedicated classes
@import 'orchestra-kit/dist/Notations/utility-classes/spaces';

// This will import utility classes for Margin and Padding generated in a single scss file
// All classes available are also available for each breakpoint and directly usable on className prop.
// for global style
.padding--space-3 {
  padding: $space-3;
}

// for style on large breakpoint
.large_space-3 {
  padding: $space-3;
}
```

### Staves

Staves are all components available to build interfaces.

</Box>
</Unstyled>

```tsx
import { Card } from 'orchestra-kit/dist/Staves/Card'
import { Title } from 'orchestra-kit/dist/Staves/Title'
import { Text } from 'orchestra-kit/dist/Staves/Text'

export default () => (
    <Card>
        <Title type="h1">Welcome</Title>
        <Text>Orchestra Design System</Text>
    </Card>
);
```
